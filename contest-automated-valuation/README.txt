This program is built for automated validation of the given solutions by multiple participants in a programming contest.

The solutions are located in corresponding folders participant1...participantN.

masterProgram.py is the main program and it checks other programs by calling them one by one accross all the participants.

out_cor contains the correct answers to the test samples stored in the fodler in.

The project is not yet complete and am not sure how effective this method is.. still searching for a better one. Anyone with a better idea, please do push it here at this question at stackoverflow.com
